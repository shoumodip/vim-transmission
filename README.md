# transmission.vim
Vim interface to the [transmission](https://transmissionbt.com) torrent client

## Installation
```vim
Plug 'shoumodip/vim-transmission'
```

## Usage
```vim
:call transmission#open()
```

| Key          | Description                     |
| ------------ | ------------------------------- |
| <kbd>r</kbd> | Refresh                         |
| <kbd>a</kbd> | Add torrent                     |
| <kbd>x</kbd> | Remove torrent under the cursor |
